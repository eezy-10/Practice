REQUIREMENT SHEET — Scientific Calculator (Evaluation Assignment)

1. Objective

Build a Scientific Calculator Web App using Vanilla JavaScript.

The goal is to evaluate:
	•	JavaScript fundamentals
	•	Code structure & modularity
	•	State management
	•	Ability to apply patterns appropriately
